**ARDUINO WEB PROGRAMMABLE TIMER**
A programmable Timer web controlled with up to 4 relay controlled output made with Arduino.

Reloj programable con un interfaz web
Intefaz web: Al estilo de la web de la black box
- Web principal donde se ve la fecha, el estado de los relés y el calendario (tipo matriz)
- Web de configuración: configurar las activaciones con un calendario semanal y un calendario para cada relé con pestañas
y luego otra pestaña de cofiguración de IP y luego otra de otras configuraciones.
- Posibilidad de conectar un sensor y activarlo relés con unos umbrales e histeresis

Clock option (selected from menu):
* NTP
* RTC

Default ip: 192.168.1.179
